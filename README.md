# Miku X-Blocker 🎵

No need to feel bad about scrolling Twitter! Miku's got your back!!

## What is this?
A Tampermonkey userscript that blocks X (Twitter) with a Hatsune Miku overlay,
so you stop mindlessly scrolling and do something better with your time.

## How to use
1. Install [Tampermonkey](https://www.tampermonkey.net/) on your browser
2. Create a new script and paste the code
3. Visit X (Twitter) and Miku will take care of the rest 🎮

## Requirements
- Tampermonkey browser extension
