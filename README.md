# Miku X-Blocker 🎵

No need to feel bad about scrolling Twitter! Miku's got your back!!
![screenshot](Animation.gif)
## What is this?
A Tampermonkey userscript that blocks X (Twitter) with a Hatsune Miku overlay.
Miku will ask you to play games with her instead of scrolling! 🎮

## How to use
1. Install <a href="https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo" target="_blank">Tampermonkey</a> on your browser
2. [Click here to install](https://github.com/nugget5600x/miku-x-blocker/raw/main/miku-x-blocker.user.js) the script
3. Visit X (Twitter) and Miku will appear! 🎵

## Requirements
- Tampermonkey compatible browser (Chrome, Firefox, etc.)
