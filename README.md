# Miku X-Blocker 🎵

No need to feel bad about scrolling Twitter! Miku's got your back!!
![screenshot](Animation.gif)<img src="demo.gif" width="200">
## What is this?
A Tampermonkey userscript that blocks X (Twitter) with a Hatsune Miku overlay.
Miku will ask you to play games with her instead of scrolling! 🎮

## How to use
1. Install [Tampermonkey](https://www.tampermonkey.net/) on your browser
2. Click [Click here to install](https://raw.githubusercontent.com/nugget5600x/miku-x-blocker/main/miku.user.js) to install the script
3. Visit X (Twitter) and Miku will appear! 🎵

## Requirements
- Tampermonkey compatible browser (Chrome, Firefox, etc.)
